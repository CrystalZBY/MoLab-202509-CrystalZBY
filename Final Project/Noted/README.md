# Noted
## Goal
- A mobile app that tracks not just mood, but also the life factors that impacts mood. Tracking the life factors, including activities done after wake-up, pre-sleep, how we eat & what we eat, sleep duration, can bring awareness to our body and mind. As data entries accumulates and displayed with data visualization graphs and highlights, we can gradually form a self-care system that optimizes wellbeing.Â 

## Files
- ContentView: start logging & see previous logs
- MoodTrackerView: select mood from 5 (very unpleasant, unpleasant, neutral, pleasant, and very pleasant)
- MoodSelectView: select the feelings that describes the mood
- MoodEntry & MoodStore: JSON files
- MoodFlowSheet: MoodTrackerView + MoodSelectView --> sheet in ContentView
- MoodImpactView: select the impact of the feelings & mood (not there yet)
